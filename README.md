# Synced busy light for M5stickC (NON PLUS) or M5-Core2

## Architecture: M5Stick-C (Door) + M5Core2 (Office)

A dual-device IoT system for office status management.

### Features
- MQTT Synchronization
- Remote Doorbell ("Ring", "Wait", "Access Granted")
- Visual Status (Red/Green/Orange)
- Mixed hardware support

### Build Instructions
1. Install VS Code + PlatformIO
2. Select your environment (m5stack-core2 / m5stick-c)
3. M5stickc on the door side and the M5Core2 at Office
4. Set your own wifi ssid, your wifi password and your MQTT broker IP with no user and no password.
5. DO NOT CHANGE MQTT TOPIC OR SYSTEM WILL NOT WORK
6. Build and flash the firmware
7. Enjoy the system.
